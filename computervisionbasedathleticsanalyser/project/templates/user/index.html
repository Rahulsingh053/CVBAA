Computer Vision Based Athletes Analyser


About Website


Athletes Analyser is a smart platform that grants core values to athletes who wants to accomplish their dreams in the field. The primary goal of Athletes Analyser is to reveal the hidden mistakes usually takes place while practicing, which directly leads to poor results even after putting efforts for years and years. This system lets user to input their videos shot during practicing, which later will be analyzed by the system and the output video would be made available on the user side. Using the pose-estimation algorithm, the system detects the angles of various body parts that plays keen role in the playground. Later, the user can compare the previously analyzed video with the output video and evaluate the difference between angles of both the videos.
